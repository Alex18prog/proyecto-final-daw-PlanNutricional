 NutriPlan

 🛠️ Tecnologías Utilizadas

- Backend:C#, ASP.NET
- Frontend:Html5, CSS3, JavaScript, Bootstrap 5.
- Base de Datos: MySQL, Entity Framework Core.
- Otras herramientas: Git, GitHub.

 🌟 Características Principales

•	Creación de un ecosistema integral: se ha conseguido fusionar una lógica de negocio complicada con una base de datos relacional en .NET, lo que evidencia la factibilidad de la arquitectura MVC para las aplicaciones sanitarias.
•	Administración dinámica de usuarios: el sistema de perfiles posibilita que la aplicación “reaccione” a la información física del usuario, logrando así el propósito de personalización.
•	Interfaz que es usable y responsiva: se ha logrado una experiencia de usuario (UX) fluida a través del empleo de Bootstrap5 y JavaScript, lo que hace posible la consulta de planes nutricionales en teléfonos móviles sin que se pierda la funcionalidad.
•	Persistencia y seguridad: La validación de formularios y el sistema de sesiones aseguran que los datos del usuario estén protegidos.
